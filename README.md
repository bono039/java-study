📕 Java의 정석 3판
=============
Java의 정석 3판을 기간 내 완독하기 위한 스터디입니다.

<br/>

## ✍️ 스터디 개요
* **주제** : Java
* **목표** : Java의 정석 1회독 완독
* **기간** : 9주 (2023-12-18 ~ 2024-02-18)

<br/>

##  📖 스터디 진행 방식
* 앞부분의 기초적인 부분(변수, 연산자, 조건문, 배열)은 제외하고 챕터6부터 시작합니다.
* 각자 공부한 결과물을 올리고 매주 월요일에 퀴즈를 통해 이해했는지 확인합니다.
* 각자 목표량 미달성 시, 매주 5000원의 벌금을 부과합니다.
